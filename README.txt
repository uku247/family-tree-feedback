# Family Tree Platform

A simple offline application to create and visualize your family tree.

---

## WHAT THIS APPLICATION DOES

The Family Tree Platform allows you to:

• Add family members
• Define relationships (parent, spouse)
• Visualize the family tree graphically
• Import an existing tree from JSON
• Export your tree as a JSON backup
• Receive update notifications when a new version is available
• Send feedback to the developer

---

## REQUIREMENTS

This application requires:

Java 17 or newer

If Java is not installed, download it from:

https://adoptium.net/

---

## HOW TO START THE APPLICATION

WINDOWS

1. Double-click:

familyTreeApp.bat

Your browser will automatically open the application.

MAC / LINUX

1. Open a terminal in this folder
2. Run:

./familyTreeApp.sh

The application will start and open in your browser.

---

## ACCESSING THE APPLICATION

If the browser does not open automatically, open:

http://localhost:8080

---

## ADDING FAMILY MEMBERS

1. Open the "Add Family" tab
2. Add persons with name, gender, and date of birth
3. Create relationships between persons

---

## IMPORTING A FAMILY TREE

If you already have a JSON backup:

1. Open "Add Family"
2. Use the "Upload Family Tree JSON" option
3. Select your JSON file

---

## EXPORTING YOUR FAMILY TREE (IMPORTANT)

Always keep a backup of your data.

1. Open the "Family Tree" tab
2. Click:

Export Family JSON

Save the file safely.

---

## UPDATES

The application automatically checks if a new version is available.

If an update exists, a banner will appear at the top of the screen.

---

## SENDING FEEDBACK

Click the "Send Feedback" button in the application.

This will open a GitHub issue where you can:

• report bugs
• suggest improvements
• request features

---

## DATA STORAGE

All family tree data is stored locally on your computer.

No data is sent to any external server.

---

## STOPPING THE APPLICATION

Close the terminal window where the application is running.

Or press:

CTRL + C
